A) What You Should Know as Buyer - top

You must have React Native knowledge or you must me inclined to learn it.

You also need to be able to install React Native and its dependencies on your machine. To do that I recommend you to use the React Native official installation guide here.

You must have Expo knowledge or you must me inclined to learn it.

You also need to be able to install Expo and its dependencies on your machine. To do that I recommend you to use the Expo official installation guide here.

And last, but not least, in order to modify the app template, you must have knowledge in mobile technologies such as TypeScript.

B) YARN Installation - top

To install your project dependencies, you must first install YARN on your system. From there you can go to the Installation docs.

YARN "Getting Started" page: here.

With YARN installed, you can proceed to the next step.

C) Install Project Dependencies - top

With YARN installed, you need to install the project dependencies. To do that you should open the command prompt (Windows), or terminal (Mac OS) ON YOUR PROJECT FOLDER and type the following commmand:

yarn install

D) Finish Project Dependencies (Mac OS) - top

If you are on a Mac OS, you need to finish installing your project dependencies by running the following commands on terminal (from your PROJECT FOLDER):

cd ios
pod install

E) Radio Player Feature - top

This app template was built with a radio player from expo-av plugin. Here are some details about it to get you started:

  1. Starting radio station data is inside "{projectName}/src/utils/constants.util.ts" file
  2. The AACP url (RADIO_STREAM_URL variable) can be obtained by installing an extension named Video DownloadHelper on you web browser
  3. You can customize background play settings by working on the staysActiveInBackground prop at "{projectName}/src/screens/Main.screen.tsx" file
  4. For more details about the expo-av plugin, click here

F) Localization Support - top

This app template currently has enUS language, but you can simply add more languages at the "{projectName}/src/languages" folder if you want. To do that, add a "xxYY.language.ts" file (similar to the "enUS.language.ts") and export it on "{projectName}/src/languages/languages.ts" file.

G) Project Details - top

Important informations about this app template:

  • React Native version: 0.69.4, more details here
